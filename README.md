Stadium Management System

It is a ticket buiyng app to the Stadiums.The project consists of two parts: documentation and implementation.Documentation is in the file MAS_Project_Nasirov_Adilet_s19245 Documentation
Documentation includes:

    User requirements
    Use Case Diagram
    Analytical Class Diagram
    Implementation/final class diagram
    Use case scenario (as enumeration)
    Activity Diagram for Use Case
    State diagram for a class
    Interaction diagram (sequence) for use case
    UI design 10 .Discussion of the design decisions and the effects of dynamic analysis

Implementation of the project

    The entire structure (classes with all associations, inheritances, etc.)
    Methods needed to implement the selected use case (or cases)
    Implementation of the GUI use case involves an interaction of two classes connected with an association (required target cardinality: many)
